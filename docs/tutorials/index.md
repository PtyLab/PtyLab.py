# Tutorials

Hands-on Jupyter notebooks covering CPM and FPM reconstruction from simulation and real data.

## In this section

- [**01 – CPM Simulation**](tutorial_CPM_sim.ipynb) — Simulate a CPM dataset and reconstruct it end-to-end.
- [**02 – CPM Example Data**](tutorial_CPM.ipynb) — Reconstruct a real CPM dataset using `easyInitialize`.
- [**03 – CPM Multi-Wavelength**](tutorial_CPM_multiWavelength.ipynb) — Handle datasets acquired with multiple illumination wavelengths.
- [**04 – CPM Position Correction**](tutorial_LateralPositionCorrection.ipynb) — Correct lateral probe-position errors during reconstruction.
- [**05 – CPM Reflection**](tutorial_reflectionPtychography.ipynb) — Reconstruct data from a reflection-mode ptychography setup.
- [**06 – CPM Interferometric Intensity**](tutorial_interferometricIntensityConstraint.ipynb) — Apply an interferometric intensity constraint during reconstruction.
- [**07 – FPM**](tutorial_FPM.ipynb) — Reconstruct a Fourier ptychography dataset from LED-array images.
